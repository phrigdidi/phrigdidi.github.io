css media queries

how does a css media query work, and why is it useful?

a css media query lets you apply different styles based on the device or viewport displaying your webpage. this is called responsive design, and it makes sure your site looks good on phones, tablets, and desktops.

for example, this code changes the layout only if the screen is at least 768 pixels wide:

@media (min-width: 768px) {
    .container {
        flex-direction: row;
    }
    .content {
        flex: 1;
        padding: 1rem;
        text-align: center;
    }
}

media queries are useful because they help you create websites that adapt to different screen sizes and devices.

what "media features" are supported by media queries?

feature example what it does
width @media (max-width: 600px) applies styles if the viewport is up to 600px wide, useful for mobile layouts.
orientation @media (orientation: landscape) applies styles if the device is in landscape mode (width > height).
hover @media (hover: hover) applies styles if the device supports hover (like desktops, not touchscreens).
prefers-color-scheme @media (prefers-color-scheme: dark) applies styles if the user prefers a dark color scheme in their os settings.
